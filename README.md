# 🛠️ Bash Scripting Lab - Dimas Teknologics

Repositório dedicado à automação, gerenciamento de sistemas Linux e ferramentas de rede. Scripts desenvolvidos para otimizar o fluxo de trabalho e garantir que a infraestrutura rode sem falhas.

> *"Voz ativa, sou mais um sobrevivente..."*

## 📌 Sobre o Repositório

Este projeto reúne scripts Shell (Bash) focados em:
- Administração de sistemas (RHEL/Ubuntu).
- Automação de tarefas repetitivas.
- Monitoramento e verificação de conectividade.
- Configurações rápidas de ambiente DevOps.

## 🚀 Como usar

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/dimas-albertini/bash-scripts.git
    ```
2.  **Dê permissão de execução:**
    ```bash
    chmod +x script_name.sh
    ```
3.  **Execute:**
    ```bash
    ./script_name.sh [argumentos]
    ```

## 📜 Scripts em Destaque

- `sys_check.sh`: Coleta métricas de CPU, memória e disco.
- `network_diag.sh`: Ferramenta rápida de diagnóstico de latência e saltos.
- `update_manager.sh`: Automatiza o update/upgrade em distros Debian e RHEL.
